로컬스토리지

        브라우저 종속 데이터베이스
        (사이트마다 구분된 공간을 가짐)
        정확히는 origin
        http://192.168.20.157:5500/
        프로토콜까지

        창을 닫았다가 다시 열어도 데이터가 유지된다
        (단, 시크릿모드에서는 불가능)
        
        Key - Value 로 구성되어 있다
        (Key, Value 는 문자열 타입이다)

        Key는 중복 불가능
        (같은 Key로 할당하면, 마지막 값으로 덮어쓰기됨)
        

        [사용법]
        * 값 셋팅하기
        localStorage.setItem(Key, Value);
        * 값 가져오기
        localStorage.getItem(Key);

JSON

        JSON : javaScript Object Notation
        
        자바스크립트의 객체 표현 방식
        JSON.parse(문자열); //문자열을 읽고, JSON 형식에 맞게 객체 생성하여 리턴
        JSON.stringify(객체); //객체를 문자열로 변환(만들어진 문자열은 JSON형식)

        [JSON 문자열]
        {"Key" : Value, "Key" : Value,...}
