객체

객체 선언

중괄호{}를 사용해서 객체를 생성하고 속성에는 모든 자료형이 올 수 있다.
객체는 속성명을 사용해서 속성을 식별한다.
속성명에 공백이나 특수문자 등이 포함된 경우 따옴표로 묶어서 표기해야 한다.
또한 객체에서 접근 시 무조건 대괄호를 사용해야 객체의 속성에 접근이 가능하다.

객체의 메소드

객체의 속성 중 함수 자료형인 속성을 메소드라고 한다.

객체와 반복문

객체가 가지고 있는 모든 속성들에 순차적으로 접근하려면
단순 for 문으로는 불가능하다. 따라서 for in 문을 사용해야 한다.

객체의 속성 추가 및 제거

객체를 생성한 이후에 새로운 속성을 추가하거나 제거할 수 있다. (동적으로 속성을 추가/제거)

객체 배열

객체와 배열을 사용해서 데이터를 관리할 수 있다.

생성자 함수

new 키워드를 사용해 객체를 생성할 수 있는 함수를 의미한다. 함수를 만들면, 프로토타입 객체가 만들어진다 (화살표 함수 제외) 프로토타입 객체는, constructor property를 가지고 있다. constructor는 프로토타입 객체를 생성한 함수를 가리킨다

캡슐화

생성자 함수에 속성을 정의할 때 this. 키워드를 사용하지 않고 지역변수로 선언할 경우
캡슐화된 객체의 속성은 직접 접근할 수 없으므로 간접적으로 접근할 수 있도록 getter/setter 메소드를 정의해야 한다.

프로퍼티 property

객체 내부의 멤버변수나 멤버 메서드를 자바스크립트에선 프로퍼티라고 부른다.
